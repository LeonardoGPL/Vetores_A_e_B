# Vetores_A_e_B
Exercício que faz a soma de dois vetores e exibe o valor da variável C
------------------------------------------------------
# ALGORITMO DO EXERCÍCIO

Inicio

    1. x == 0
    2. x == x+1
    3. A[x],B[x]
    4. Se x > 4 
    5. x == 1
    6. C == 0
    7. Fim enquanto
    8. Se x <= 5
    9. C = C + A[x] * B[x]
    10. x = x + 1
    11. Exibe o valor de C

Fim
